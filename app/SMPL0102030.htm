Video Template

This page contains an example of a video. Videos should be in MP4 format and use MediaElement.js player to ensure compatibility across various browsers. The video must also contain both Closed Captioning (.srt file) and a transcript.

Use the source code of this page "SMPL0102030.htm" as a template for videos. You will need to make the following code changes:

  • Poster attribute of the video tag
  • src attribute of the source tag
  • value attribute of the param tag with the name "flashvars"
  • Track tag when using a separate caption file
  • Link to transcript

This page also contains an example of an internal link. Internal links can be used for various reasons including linking to additional content, printable materials, and transcripts.
