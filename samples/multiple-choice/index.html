Knowledge Review (Checkbox/Multiple Select)

This is an example of a knowledge review that contains checkboxes. Users can select multiple boxes. A reference to the file: "FeedbackCA.js" is required. The following JavaScript variables must also be changed: Feedback and vCorrectAnswer.

The EMI Snippet titled "CheckAllThatApply" may be used as a starting point for any multiple choice, multiple answer style question.

Example:

Instructions: Select all items that apply. Click on the Submit button when you are finished.
