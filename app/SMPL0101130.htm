Page Templates

The source materials for this course contains a variety of templates that should be used when developing EMI courses. The templates can be found under the "templates" directory and are Dreamweaver files with the ".dwt" extension.

We provide different templates for different types of slides/page content. For example, if you would like to have a slide with video content, you would use our video template. Example content can be found in lesson 2 of this course. Each example in that section was created with the appropriate template.

Using the Templates

You may use Dreamweaver to create files from templates; however, this is not required. Simply follow the steps below:

  1. Open the appropriate template (.dwt) in the html editor of your choice.
  2. Look for all editable areas - Editable areas are marked by the following HTML comment tags"<!-- TemplateBeginEditable name="PageHeader" -->" and <!-- TemplateEndEditable -->" where "PageHeader" may differ.
  3. Remove content in all editable areas and replace with new content that should be displayed.
  4. Save as (.htm) file with appropriate name.
