Knowledge Review (True/False)

This page contains an example of a "True/False" Knowledge Check. Follow the example found on this page. The "FeedbackMC.js" must be referenced for this to work properly. In addition, the following JavaScript variables should be modified: Feedback, FeedbackWrong, and vCorrectAnswer.

The EMI Snippet titled "MultiChoice" can be used as a basic template for the True/False question or any multiple choice, single answer question.

Example:

Instructions: Review the case study and select your answer.  

Case Study: The Lummi Nation has a clinic that effectively serves the tribal communities for day-to-day and routine health care issues. Although the clinic is responsible for public health in the community, it lacks the resources to respond to a public health emergency. For example, the tribe does not have a mechanism in place for quarantine and isolation.

True or False: the Lummi Nation's clinic is prepared for public health emergencies.
