Required Content

EMI requires that certain "screens" be included in every course. As you create a course via the EMI template Ensure that you include every "screen" listed below. Examples of each of these can be found within this course.

  • Course Overview
    • "Lesson List" image required - use source/lessonList.psd
  • Screen Features*
  • Navigating Using Your Keyboard*
  • Receiving Credit*
  • Lesson Overview
    • "Lesson List" image required - use source/lessonList.psd
  • Lesson Summary (For Each Lesson)
    • "Lesson List" image required - use source/lessonList.psd
  • Printable Full Course
  • Printable Individual Lesson Summaries
  • Final Exam page

* - Designates pages that should be pulled word-for-word from this template. These "screens" do NOT need to be changed and must be included in your course.
