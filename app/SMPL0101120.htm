Editing the ismanifest.xml File

The ismanifext.xml file must be properly structured and include specific references for the course to function properly. This is required for to create a SCORM compliant course. Follow the example found within this course and use the following information to update your ismanifest.xml file.

This XML file is broken down into various required nodes. Place the appropriate information at each node path shown below:

  • Title: Manifest -> Metadata -> imsmd:lom -> imsmd:general -> imsmd:title ->imsmd:langstring
  • Description: Manifest -> Metadata -> imsmd:lom -> imsmd:general -> imsmd:description ->imsmd:langstring
  • Keywords (multiple allowed): Manifest -> Metadata -> imsmd:lom -> imsmd:general -> imsmd:keywords ->imsmd:langstring
  • Course Title (again): Manifest -> Organizations -> Organization -> Title
  • Lessons (multiple allowed): Manifest -> Organizations -> Organization -> Item
  • Lesson Pages (multiple allowed): Manifest -> Resources -> Resource -> File

Note: For each lesson you will have a resource and multiple files underneath of that resource. Each individual file represents a page/slide within that lesson.
